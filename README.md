# 麻雀成績計算アプリ

対局後の最終持ち点を入力して、順位ウマとオカ込みのポイントを計算・記録するWebアプリです。

## 機能

- 通常モードと大会モード
- 4麻 / 3麻の切り替え
- 配給原点、返し点、順位ウマの設定
- オカは配給原点と返し点から自動計算
- 複数半荘の合計ポイント表示
- 大会モードでは5人以上の参加者から卓のメンバーを選択
- ブラウザの `localStorage` に自動保存

## 計算式

```text
オカ = (返し点 - 配給原点) * 卓人数 / 1000
ポイント = (最終持ち点 - 返し点) / 1000 + 順位ウマ + オカ
```

オカは1位だけに加算されます。
